# Protoss Tech Tree

```
Nexus
├── Pylon
├── Assimilator
└── Gateway
    ├── Forge
    │   └── Photon Cannon
    └── Cybernetics Core
        ├── Shield Battery
        ├── Twilight Council
        │   ├── Templar Archives
        │   └── Dark Shrine
        ├── Robotics Facility
        │   └── Robotics Bay
        └── Stargate
            └── Fleet Beacon
```

## Three independent branches from Cybernetics Core

1. **Twilight path** — Twilight Council → Templar Archives / Dark Shrine
2. **Robo path** — Robotics Facility → Robotics Bay
3. **Air path** — Stargate → Fleet Beacon

These are parallel, not sequential. A bot can pursue any combination.
